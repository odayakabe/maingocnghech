Giới thiệu phần mềm Markdown soạn thảo văn bản nội dung cho open source chạy trên GitHub

Giới thiệu phần mềm Markdown có preview xem trước dễ dàng soạn thảo văn bản cho những định dạng bài viết dạng .md trên các open source phổ biến chạy trên GitHub hiện nay như Jekyll, Hugo, Hexo … vân vân… xem thêm nhiều open source khác ở StaticGen.

Kobito

Thực ra thì Kobito là phần mềm Markdown được phát triển riêng cho thị trường Japan - Nhật Bản. bởi cộng đồng Open source và phát triển phần mềm Qiita. Tạm gọi là Kobito Markdown Software của Qiita.

Kobito là phần mềm Markdown có Preview xem trước nội dung soạn bằng ngôn ngữ Markdown phổ biến bởi định dạng .md của các open source chạy trên GitHub như giới thiệu ở trên.

Kobito có nhiều phiên bản khác nhau cho các phần mềm như: Windows, Mac O.S.

Screenshot

Tải về Kobito Markdown

Khi truy cập vào Kobito thì hệ thống này tự động nhận dạng hệ điều hành của bạn mà đưa ra phiên bản phần mềm hợp lý.

Ở đây, Thảo Am sử dụng hệ điều hành Windows 7 cho nên Qiita Kobito nó giới thiệu luôn cho phiên bản Kobito for Windows.

Khá tiện lợi.

Soạn thảo Markdown bằng Kobito

Ngay khi tải về và cài đặt phần mềm Kobito này, đã có ngay hướng dẫn viết Markdown tại những bài viết mặc định của Phần mềm Kobito, như hình dưới đây

Quá dễ dàng, chỉ cần căn cứ theo bảng hướng dẫn và điền ký tự phù hợp khi viết là đã tạo ra một bài viết với định dạng .md hoàn chỉnh mà không phải đắn đo nhiều.

Mẫu bài viết Markdown với Kobito xem trước

Đây cũng là bài viết mà Thảo Am sử dụng để giới thiệu phần mềm Kobito này, các bạn xem hình dưới đây để tham khảo nhé.

Sau khi viết bài xong, mình điền thêm Tag cho bài viết rồi Save lại.

Thích thì mở Preview ra xem trước bài đăng xem hình dáng thế nào để còn sửa, xong xuôi rồi thì chỉ việc Save lại dưới định dạng: tên-bài-viết.md là xong rồi copy vào folder bài viết của Open source của mình là xong.
